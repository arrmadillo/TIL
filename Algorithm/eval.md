# 큰 수식 찾기( eval() )

## 문제

```jsx
//입력
10+5 10+6

//출력
16

//입력
10*3-10 10*4-50

//출력
20
```

## eval() 함수

- 이 함수는 인자로 주어진 문자열이 수식이라면, 수식의 결과를 반환해주는 함수

```python
A = "10*10-99"
B = "10-99*10"
print(eval(A))
print(eval(B))

'''
출력
1
-980
'''
```

## 정답

```python
A, B = input().split()
print(max(eval(A), eval(B)))

# 각 수식을 계산 후 max함수로 최댓값 출력
```
