# pip (파이썬 패키지 관리시스템)

## 명령어

### 패키지 설치

```
pip install 패키지이름
pip install 패키지이름==1.0.5  # 특정 버전 설치
pip install '패키지이름>=1.0.5' #최소 버전 설치
```

### 패키지 삭제

```
pip uninstall 패키지이름
```

### 패키지 업그레이드

```
pip install --upgrade 패키지이름
```

### 버전확인

```
pip --version # pip 버전확인
```

### pip 업그레이드

```
pip install -U pip #리눅스 or mac
python -m pip install --upgrade pip #윈도우
```

### 설치된 패키지 목록 확인

```
pip list
```
