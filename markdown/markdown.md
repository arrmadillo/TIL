# 간략한 마크다운 요약!

## 마크다운

- ### 정의

  마크다운은 일반 텍스트 기반의 경량 `마크업` 언어다. 일반 텍스트로 서식이 있는 문서를 작성하는 데 사용된다.

- ### 간략한 문법

### `제목`: 1 ~ 6까지 사용가능

```html
# 제목 1 ## 제목 2 ### 제목 3 #### 제목 4 ##### 제목 5 ###### 제목 6
```

### `강조`

```
`내용`
```

### `순서 있는 리스트`

```
1. 내용
2. 내용
```

### `순서 없는 리스트`

```
- 내용
- 내용
or
* 내용
+ 내용
```

### `볼드`

```
**내용**
__내용__
```

### `코드첨부`

````
```(표시언어)
소스코드작성
```(끝)
````
